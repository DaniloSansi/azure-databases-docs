### intelligent_tuning

| Attribute      | Value                                                      |
|----------------|------------------------------------------------------------|
| Category       | Intelligent Tuning |
| Description    | Enables intelligent tuning                                      |
| Data type      | boolean   |
| Default value  | `off`         |
| Allowed values | `on,off`                                                                                                                           |
| Parameter type | dynamic        |
| Documentation  | [intelligent_tuning](https://docs.microsoft.com/azure/postgresql/flexible-server/concepts-intelligent-tuning)                |

### intelligent_tuning.metric_targets

| Attribute      | Value                                                      |
