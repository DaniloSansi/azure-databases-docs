# Azure Cosmos DB: MongoDB API documentation

Azure Cosmos DB has the following documentation that's specifically for creating document databases with the MongoDB API.

These articles do not apply to the DocumentDB API, Graph API, or Table API. 

## Introduction and concepts

These are must read topics and resources to start learning about the DocumentDB API for Azure Cosmos DB.

- [MongoDB API introduction](mongodb-introduction.md). Provides an overview of the capabilites of the MongoDB API.
- [Connect to your MongoDB account](connect-mongodb-account.md). Learn how to connect your MongoDB app to an Azure Cosmos DB account by using your MongoDB connection string. 

## Quickstarts

The Quickstart topics are the fastest way to create a working application with Azure Cosmos DB. In each Quickstart you'll learn how to use the UI-based Azure portal and your favorite coding language to create database solutions with Azure Cosmos DB. Cloneable web apps from GitHub area available for each Quickstart. 


- [MongoDB + Express + AngularJS + Node.js + Web apps](create-mongodb-nodejs.md)
- [.NET + Azure Portal + Web apps](create-mongodb-dotnet.md)
- [Java + Azure portal + Web apps](create-mongodb-java.md)
- [Go + Azure portal + Goglang](create-mongodb-golang.md)

## Tutorials

These tutorials are a level deeper than the Quickstarts. In the tutorials you'll build apps from scratch and copy and paste code into the app. You'll also learn how to import data, query data, and distribute your databases globally.

### Tutorials to create a console app

These tutorials walk you through the creation of a console app that uses Azure Cosmos DB to store data.  

- [Node.js](mongodb-samples.md) 
- [.NET](tutorial-develop-mongodb.md)

### Tutorial to migrate your MongoDB data

This tutorial walks you through the process of importing MongoDB data to Azure Cosmos DB.

- [Import MongoDB data](mongodb-migrate.md)

### Tutorials to work with your data

- [Import data](import-data.md). This tutorial shows you how to use the Azure Cosmos DB: DocumentDB API Data Migration tool to import data from various sources, including JSON files, CSV files, SQL, MongoDB, Azure Table storage, Amazon DynamoDB and Azure Cosmos DB DocumentDB API collections into collections for use with Azure Cosmos DB and the DocumentDB API.
- [Query](tutorial-query-mongodb.md). A brief introduction to query basics using the MongoDB API.
- [Distribute data globally](tutorial-global-distribution-mongodb.md). Learn how to distribute data globally using the MongoDB API.
